# Site Institucional — Grupo VanguardIA

Página institucional do **Grupo VanguardIA**. Single-page em HTML/CSS/JS puro, sem build step, com narrativa épica e arquitetura visual premium.

## Estrutura

```
vanguardia-site/
├── index.html         # Página única, autocontida
├── CLAUDE.md          # Briefing institucional do projeto (contexto para Claude Code)
├── README.md          # Este arquivo
├── .gitignore
└── assets/
    ├── logo-avatar.png
    ├── logo-horizontal.png
    └── logo-vertical.png
```

## Rodando localmente

Não precisa instalar nada. Abre o `index.html` no navegador, ou serve com:

```bash
python3 -m http.server 8000
# ou
npx serve .
```

Depois acessa `http://localhost:8000`.

## Deploy

O projeto é static site puro. Funciona em Vercel, Netlify, GitHub Pages, ou qualquer CDN.
